# Simple Calculator

This small Python project provides three simple functions in `calc.py`:

- `create(value=0)` — validate and return a numeric value
- `add(a, b)` — return the sum of two numbers
- `subtract(a, b)` — return the difference

Usage:

Run the demo CLI:

```powershell
python main.py demo
```

Add/subtract from the command line:

```powershell
python main.py add 3 4
python main.py sub 10 7
```

Run tests (install pytest if needed):

```powershell
pip install pytest
pytest -q
```
